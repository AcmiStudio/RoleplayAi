# 🎭 RoleplayAI — сетевая текстовая RPG с ИИ-рассказчиком

**Первая стабильная версия (v1.0.0)**  
_Русскоязычное приложение (интерфейс и игровой процесс ориентированы на русский язык)_

RoleplayAI превращает текстовую ролевую игру в живое приключение под управлением локального ИИ (Ollama). Создайте комнату, позовите друзей (до 4 игроков) и погрузитесь в мир, который придумывает и описывает искусственный интеллект. Всё работает в консоли, кроссплатформенно (Windows, Termux на Android) и не требует подключения к интернету после установки моделей.

---

## ✨ Основные возможности

- **🧠 ИИ-рассказчик**  
  Локальная модель Ollama генерирует сюжет, описывает окружение и отыгрывает NPC. Вы описываете действия персонажа — ИИ реагирует как настоящий ведущий.

- **👥 Многопользовательская игра**  
  Один администратор создаёт комнату, до 4 игроков подключаются по локальной сети (WebSocket). Играйте с друзьями за одним столом или по вайфаю.

- **🎛 Два режима общения**  
  - *Свободный* – все игроки действуют одновременно, ИИ отвечает на общий контекст.  
  - *Поочерёдный* – строгая очерёдность ходов, как в классических настольных RPG.

- **🎨 Цветовое разделение игроков**  
  Каждый участник может выбрать свой цвет, чтобы реплики в консоли было легко различать.

- **✍️ Встроенный редактор персонажей и миров**  
  Создавайте, редактируйте и сохраняйте описания персонажей и миров прямо из меню. Вся информация хранится локально в текстовых файлах (папки `data/characters` и `data/worlds`).

- **🕹 Административные команды**  
  Управляйте игрой из консоли сервера или клиента-администратора:  
  `/start`, `/stop`, `/kick`, `/jail`, `/unjail`, `/mode`, `/help`.

- **💾 Гибкие настройки**  
  - Выбор модели Ollama, температуры генерации.  
  - Загрузка кастомного промпта рассказчика из файла или ручной ввод.  
  - Лёгкое переключение персонажа и цвета через меню.

- **🖥 Кроссплатформенность**  
  Полностью работоспособно на Windows (консоль) и в Termux (Android). Чистый Python, минимальные зависимости.

## 📋 Системные требования

- **Python** 3.10 или выше.
- **Размер** ≈ 1-2 GB(Если использовать Termux)
- **Ollama** (установленная и запущенная) с любой совместимой моделью (рекомендуется `gemma4:31b-cloud` или `llama2`).
- Пакеты Python: `websockets`, `colorama`, `aiohttp` (устанавливаются автоматически через `setup.sh` или `pip`).
- Локальная сеть для подключения игроков (или один компьютер для теста).

## 🚀 Установка и запуск

### 🐧 Termux (Android)

1. Установите Termux из F-Droid или GitHub.
2. Введите готовую команду:
```bash
pkg update -y && pkg upgrade -y && pkg install git && git clone https://github.com/AcmiStudio/RoleplayAi.git && ls && cd RoleplayAi
```
3. Перейдите в папку с игрой и выполните:

```bash
chmod +x setup.sh
./setup.sh
```
(При пункте 3 у вас будут задавать вопросы по типу (N/D/S/O/Y) что то вроде этого.. это если что выбор, ваша задача просто писать "Y" или "y " без кавычек.)
Скрипт сам обновит пакеты, установит Python и необходимые библиотеки, создаст структуру папок и базовый файл настроек.

1. Запустите игру:

```bash
python main.py
```

Примечание: Ollama в Termux может работать нестабильно. Рекомендуется запускать сервер Ollama на ПК и указать его IP в коде (в server.py, заменив localhost на адрес ПК). В планах – автоматический конфиг для удалённого сервера.



---

⚙️ Настройка Ollama

Перед игрой убедитесь, что Ollama запущена и у вас загружена хотя бы одна модель. Например:

```bash
ollama serve          # запустить сервер (обычно уже работает как служба)
ollama pull gemma4:31b-cloud   # загрузить модель
```

В меню игры выберите Настройки → Настройки модели Ollama, чтобы указать имя модели, температуру и промпт рассказчика.

---

🎮 Как играть

1. Создайте комнату (пункт 1 в главном меню).
      Укажите название, количество игроков, режим общения. Затем выберите готовый мир из файла, создайте новый или введите описание с клавиатуры.
      Сервер запустится и покажет локальный IP для подключения.
2. Подключитесь к комнате (пункт 2).
      Игроки вводят свой адрес (ws://IP:8765), имя персонажа и готовое описание (или создают персонажа через редактор). При подключении цвет игрока берётся из настроек.
3. Начните игру – администратор в консоли сервера (или через клиент-администратора) вводит команду /start.
      ИИ-рассказчик генерирует вступительную сцену, и игроки по очереди (или свободно) описывают действия.
4. Управляйте игрой командами:
   · /start – запустить игру.
   · /stop – завершить.
   · /kick <имя> – отключить игрока.
   · /jail <имя> – временно заблокировать (сообщения игнорируются).
   · /unjail <имя> – разблокировать.
   · /mode free или /mode turn – сменить режим общения.
   · /help – помощь.

📂 Структура проекта

```
RoleplayAI/
├── main.py                 # главное меню и запуск
├── server.py               # серверная часть (WebSocket + Ollama API)
├── client.py               # клиент для игроков
├── config.py               # работа с настройками, персонажами и мирами
├── utils.py                # консольный интерфейс и цвета
├── requirements.txt        # зависимости (если не используется setup.sh)
├── setup.sh                # автоустановка для Termux
└── data/
    ├── settings.json       # конфигурация (модель, цвет, промпт и т.д.)
    ├── characters/         # ваши персонажи (.txt)
    └── worlds/             # ваши миры (.txt)
```
